# Todo API

A simple Todo API built with NestJS and PostgreSQL.

## Getting Started

### Installation

```bash
npm install
```

### Setup Environment

Create a `.env` file with your database configuration:

```env
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=postgres
DB_PASSWORD=postgres
DB_DATABASE=todo_db
```

### Run the App

```bash
npm run start:dev
```
